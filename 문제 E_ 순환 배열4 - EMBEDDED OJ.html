문제 E: 순환 배열4숨겨짐

메모리제한：256 MB 시간제한：1.000 S
체점 스타일：텍스트 비교 만든사람：관리자 업로드
제출：93 통과：1
대회로 돌아가기제출채점기록제출(듀얼모드)

문제 설명

unsigned int 100칸 배열 arr에서, 다음과 같은 방법으로 순회하는 프로그램을 작성하시오.

1. 시작은 인덱스 0이다. 인덱스는 바이트 단위로 움직인다.
2. 해당 인덱스에 있는 값(arr[i])에서, 비트의개수비트의개수(비트1의개수)−(비트0의개수)만큼 이동한다.
3. 이동이란, 해당 값만큼 바이트 단위로 이동하는 것이다.
4. 단, 배열의 구역을 넘어가면, 반대쪽으로 돌게 된다. 만약 이동 결과, 바이트의 일부가 배열의 바깥을 가리킨다면, 해당 바이트는 반대쪽의 값을 이용한다.
4-1. 만약 인덱스가 399이면, 399, 0, 1, 2 번째 바이트를 이용하는 것이다.
5. K번의 이동을 한다.

arr의 값을 입력하면, 위 방법에 따라 순회한 결과를 출력하는 프로그램을 작성하시오.

입력 설명

첫째 줄에 K를 입력한다. (1≤K≤100)
둘째 줄에 arr에 저장할 값 100개를 공백으로 구분하여 입력한다. 각 값은 32bit 대문자 16진수이다.

출력 설명

각 이동마다 도착하는 인덱스 K개를 공백으로 구분하여 출력한다.
